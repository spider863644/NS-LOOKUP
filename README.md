## Installation for Linux
<b>Open Linux and type the following command below</b>
```
sudo apt update
sudo apt install python3
sudo apt install git
git clone https://github.com/spider863644/NS-LOOKUP
cd NS-LOOKUP
python3 nslookup.py


```
